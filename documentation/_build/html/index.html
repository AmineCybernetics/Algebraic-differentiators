discretize(der, method='mid-point', redFilLength=False, redTol=0.01, discreteSpectrum=False)¶

This function performs the discretization of the filter for the estimation of the derivative of order der. Three discretization schemes are implemented: midpoint, trapezoidal and using the analytical integration. The error stemming from the discretization if corrected using a correction factor.

Parameters:
  • der (int) – order of the derivative to be estimated
  • method (string) – discretization scheme
  • reduceFilterLength (bool) – reduce or not the filter window length
  • redTol (float.) – tolerance to be used when the filter length is reduced
Returns:

discretized filter
Return type:

numpy array
estimateDer(k, x, method='mid-point', conv='same', redFilLength=False, redTol=0.01)¶

This function estimates the derivative of order \(k\geq0\) using the discretization method given. If the filter was not discretized before the function performs the discretization.

Parameters:
  • k (int) – order of the derivative to be estimated
  • x (numpy array) – signal whose derivative has to be estimated
  • conv (string) – parameter to specify the type of convolution the numpy conv function does
  • reduceFilterLength (bool) – reduce or not the filter window length
  • redTol (float.) – tolerance to be used when the filter length is reduced
Returns:

estimated derivative
Return type:

numpy array of same size as x

get_ratioNyquistCutoff(k)¶

This function computes the ratio \(k_N=\frac{\omega_N^k\big|\mathcal{G}(\omega_N)\big|}{\omega_c^k\Big|\mathcal{G}(\omega_c)\Big|}\), with \(\omega_N=\pi/t_s\), the Nyquist frequency and \(\omega_c\) the cutoff frequency.

Parameters:k (list of natural numbers) – order of derivative to be estimated
Returns:\(k_N\)
Return type:numpy array with unit dB and length=len(k)
